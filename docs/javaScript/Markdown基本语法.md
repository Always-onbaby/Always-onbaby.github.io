#### 代码

如果你只想高亮语句中的某些关键字或者函数名，可以使用反引号`function_name`实现。  
通常编辑器会根据代码片段匹配适合的高亮方法，也可以使用 ` 包裹代码，指定语言

```javascript
${document}.ready(function() {
    alert('hello world')
})
```

```javascript

#### 小标题
```
